# Claude Experimenting

A collection of projects built while experimenting with Claude AI.

## Projects

### [custom-cms](./custom-cms)
A full-featured CMS built with Next.js, Drizzle ORM, and Claude AI. Includes an admin dashboard, REST API, and public-facing site with pages, blog posts, and products.

**Stack:** Next.js 16, React 19, TypeScript, SQLite, Tailwind CSS 4, Claude AI

### [pig-dice](./pig-dice)
A vanilla JavaScript implementation of the Pig dice game with multi-player support, AI bots, and stat tracking.

**Stack:** HTML, CSS, JavaScript
